# ammolite.css
## About

Minimal CSS/HTML5 boilerplate made for starter projects.

## What you get

* Everything reseted & normalized
* Typography
* Basic CSS classes
* Control through variables

## Variables

You can find variables at top of "./assets/ammolite.css" file
