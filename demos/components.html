Web components

ammolite.css (for now) comes with two components. Radio and checkbox.

Radio buttons

This is basic radio button component that looks like this: 

<amm-radio> 
    <slot selected>Alpha</slot>
    <slot>Beta</slot>
    <slot>Gamma</slot>
</amm-radio>

And like this when processed:

Alpha Beta Gamma

To get which item is selected, run 'getElementState' method like this: 

    var item = document.getElementById('new-radio');    
    console.log( item.getElementState() );

Checkbox

This is basic checkbox component that looks like this: 

<amm-checkbox id="new-checkmark"> 
    <slot disabled> Alpha</slot>
    <slot> Beta</slot>
    <slot> Gamma</slot>
    <slot selected> Theta</slot>
</amm-checkbox>

And like this when processed:

Alpha Beta Gamma Theta

To get which item is selected, run 'getElementState' method like this: 

    var item = document.getElementById('new-checkbox');    
    console.log( item.getElementState() );
